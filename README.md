# Expenses App

Aplicação que exemplifica o uso de gráficos no React Native.

## Stack

- React Native
- Typescript
- Expo 
- Victory Charts
- React Native SVG


## Recursos

- Filtrar despesas por mês
- Exibir o gráfico de pizza


## Execução

Clone this project and start with Expo: 
```bash
  git clone branch_url
  cd expenses-app
  yarn
  yarn start
```
